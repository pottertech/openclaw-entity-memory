# KNOWN-LIMITATIONS.md

## Purpose

Document known limitations of outage-impact hybrid memory at the end of Phase 10.

## Categories

### Data limitations
- source adapter gaps
- incomplete owner mappings
- incomplete dependency mappings

### Retrieval limitations
- semantic baseline quality limits
- evidence visibility limits
- missing evidence cases

### Graph limitations
- in-memory backend still primary or Kuzu not yet fully active
- conflict resolution remains heuristic in some cases

### Governance limitations
- ACL inheritance may still be simple
- document visibility may hide useful evidence
- some exclusions may still require operator interpretation

## Rule

Known limitations must be recorded openly.
Do not hide them to justify promotion.
